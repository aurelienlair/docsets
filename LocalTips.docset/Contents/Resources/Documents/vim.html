# Delete without saving into the register 

  To delete something without saving it in a register, you can use the
  "black hole register".

  The following command:
  - deletes inside something inside "<" (eg. "<title>Hello World</title>") 
  - puts it into the black hole register
  - pastes what is in the buffer

  aurelien@linux:~$ "_di>P

  Same thing but deleting until ">"
  aurelien@linux:~$ "_dt>P

#  Jumping to previous location 

  To jump to the previous location run the following command.
  aurelien@linux:~$ Ctrl-o

# To remove the current file from the file system with vim use `%` (% contains the name of the current file with the full path) 

  Command within vim:
  aurelien@linux:~$ :!rm %

  `!` will invoke bash while `%` is the full file path 
  (like /home/aurelien/filename.php)
