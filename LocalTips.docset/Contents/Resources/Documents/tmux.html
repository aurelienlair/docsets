# Ctrl+space : allow you to run commands throw Tmux's prompt 

     aurelien@linux:~$ ctrl+space : find-window Vim

     is the same as `ctrl+f` "Vim"

# Ctrl+space ? list the shortcuts available with Tmux 

     aurelien@linux:~$ ctrl+space ?

# Ctrl+space ] paste buffer content throw Tmux with vim 

     aurelien@linux:~$ ctrl+space ]

# Ctrl+space [ start copy mode throw Tmux with vim 

     aurelien@linux:~$ ctrl+space [

# Ctrl+space # in Tmux will display the list of buffers 

     aurelien@linux:~$ ctrl+space #

# Ctrl+space . move current window. It is necessary to insert a new number to the prompt (should not be already used) 

     aurelien@linux:~$ ctrl+space . 3

# Ctrl+space = lists all paste buffers and by choosing the one interested with ↑ and ↓ pastes selected buffer contents with Tmux 

     aurelien@linux:~$ ctrl+space =
