# Ctrl+w cut the word before the cursor. If it's a path it cuts the path 

     aurelien@linux:~$ cd /home/aurelien

     will become

     aurelien@linux:~$ cd

# Ctrl+u cut all the words before the cursor 

     aurelien@linux:~$ cd /home/aurelien

     will become

     aurelien@linux:~$

# Ctrl+y will yank the buffer content from the cursor's position 

     Example with `aurelien` in the buffer with the cursor after /

     aurelien@linux:~$ cd /home/

     will become

     aurelien@linux:~$ cd /home/aurelien

# Alt+b will move back the cursor at the begining of the previous word 

     If the cursor is at the end in this example

     aurelien@linux:~$ cd /home/aurelien

     by pressing `alt+b` it will put the cursor at the begining 
     of `aurelien`

# Alt+f will move forward the cursor after the word 

     If the cursor is at the begining in this example

     aurelien@linux:~$ cd /home/aurelien

     by pressing `alt+f` it will put the cursor after `cd`

# Alt+d will delete the characters from current cursor position til the end of the word 

     If the cursor is at the begining in this example

     aurelien@linux:~$ cd /home/aurelien

     by pressing `alt+d` it will remove `cd` 

     aurelien@linux:~$ /home/aurelien
